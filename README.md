push 步驟

1. git add .  ->  將修改推至本機端倉庫
2. git commit -m "修改註解" -> 將更動提交至Head
3. git checkout -b 自己的branch -> 創建一個自己的branch，並切換過去
4. git push origin 自己的branch -> push到遠端gitlab
